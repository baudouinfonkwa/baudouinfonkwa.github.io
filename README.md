# Professional Portfolio — GitHub Pages (Jekyll)
This repo includes an interactive **Rayleigh–Plesset Bubble Solver** at `/pages/bubble-solver`.

## Quick Start
1) Create a repo named `YOURUSERNAME.github.io` on GitHub.  
2) Upload these files (or push via git).  
3) Settings → Pages → select the `main` branch.

## Customize
- Title/description: `_config.yml`
- Home: `index.md`
- Pages: `pages/*.md`
- Blog posts: `posts/YYYY-MM-DD-title.md`
- Styles: `assets/css/style.css`
- Bubble solver: `pages/bubble-solver.md`
